# STM32-Custom-Dev-Board-USB-Powered-
STM32 Custom Dev Board (USB Powered) is a minimal STM32 microcontroller development board designed for embedded firmware development and hardware prototyping. The board includes USB power input, a 3.3 V LDO regulator, external crystal oscillator, SWD/JTAG Interface boot configuration circuitry, reset switch, and GPIO LEDs for testing and debugging.
